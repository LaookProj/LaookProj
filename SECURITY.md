# Security Policy

## Supported Versions

The following versions of LaookProj components are currently supported with security updates:

| Component | Supported Versions |
| :--- | :--- |
| `laook-backend-api` | `>= 1.0.0` |
| `laook-models` | `>= 1.0.0` |
| `laook-mobile-app` | `>= 1.0.0` |
| `laook-flutter-app` | `>= 1.0.0` |

## Reporting a Vulnerability

We take the security of LaookProj seriously. If you find a security vulnerability, please do not report it publicly via GitHub issues. Instead, follow the process below:

1. **Email the Maintainers**: Send a detailed security report to `security@laookproj.org` (or contact the organization owners directly through GitHub).
2. **Information to Include**:
   - Description of the vulnerability and its potential impact.
   - Step-by-step instructions to reproduce the vulnerability (proof of concept).
   - Any proposed mitigations or fixes.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible. We ask that you adhere to responsible disclosure practices and give us reasonable time to release a patch before disclosing the issue publicly.

Thank you for helping keep LaookProj secure!
